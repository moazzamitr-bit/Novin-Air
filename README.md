# Novin Air

Luxury Persian landing page for Novin Air, built with Next.js, Tailwind CSS, Framer Motion, and Vazirmatn typography.

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## GitHub Pages

The project is configured for GitHub Pages through `.github/workflows/deploy-pages.yml`.
The deployed preview URL is:

https://moazzamitr-bit.github.io/Novin-Air/
